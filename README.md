# Radio Check Ternary

Простое приложение на C# Windows Forms с проверкой выбранного radioButton через тернарный оператор.

## Функциональность

- Два radioButton
- Кнопка для проверки выбора
- Вывод результата в textBox1
- Использование тернарного оператора

## Интерфейс

- radioButton1 — первый вариант
- radioButton2 — второй вариант
- button1 — кнопка проверки
- textBox1 — отображение результата

## Пример работы

Если выбран radioButton2 → textBox1 показывает "Верно"
Иначе → textBox1 показывает "Ошибка"

## Куда можно встроить

- В тестовые системы с проверкой ответов
- В учебные материалы по тернарному оператору
- В демонстрацию работы radioButton

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Проверяется только состояние radioButton2.
